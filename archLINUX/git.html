0. installing git using pacman -S git (python required?)
_______________________________________________________
1. DOWNLOAD PACKAGE:
--------------------
git clone https://aur.archlinux.org/package_name.git
cd package_name
-----------------------------------------------------
INSTALL THE PACKAGE:
makepkg -si
OR USE THIS (to install package):
pacman -U package.pkg.tar.xz
==================method 2 by hand======================
DOWNLOAD COMPRESSED FILE DIRECTLY BY CLICKING ON IT OR ON CONSOLE :
curl -L -O https://aur.archlinux.org/cgit/aur.git/snapshot/package_name.tar.gz
DECOMPRESS/EXPORT FILES USING THIS COMMAND:
tar -xvf package_name.tar.gz
====================================================
